== Description ==
NMedia Wordpress Member Conversation Plugin allow site users to communicate each other. Users can send messages to each other using ajax based request. The interface is simple and stylish and thus fit for every theme. Strong validation, wp core security used to purify data and stop spamming.

<h3>Features</h3>
<ol>
	<li>Ajax based conversation</li>
	<li>Autocomplete users list</li>
	<li>New conversation notification</li>
	<li>All conversations list in admin</li>
	<li>Multi Language supported</li>
	<li>Smooth pagination</li>
	<li>Strong validation</li>
	<li>WP core data validation</li>
	<li>No spamming</li>
</ol>

All is done using a simple short code: [nm-wp-convo]

<h3>Unread messages AlertBox</h3>
To display an alert about unread message anwhere in your site you can use this shortcode: `[nm-convo-alertbox pageurl="http://domain.com/convopage_url"]`

== Installation ==

1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After activation, you can set options from `NM FileUploader` menu
